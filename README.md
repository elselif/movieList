
# NETFLIX


- TMDB API was used
- To my list by categories
- Pagination Structure


## Run on Your Computer

Clone the project

```bash
  git clone https://github.com/elselif/netflixClone.git
```


Install required packages

```bash
  npm install
```

Run the server

```bash
  ng serve
```
